## WhatsApp Clone
Web based Whatsapp clone with real time private chat features.
## Tech Stack used:
- Frontend:  Reactjs, Material-ui
- Backend: Node.js and Express.js
- Database: MongoDB Atlas
- Authentication: Google Authentication
